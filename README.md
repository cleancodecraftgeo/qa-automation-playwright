# qa-automation-playwright
Web automation testing project built using Playwright and TypeScript

This repository contains UI automation tests built with Playwright and TypeScript.

The project is built for practicing real QA workflows using a demo e-commerce application.

## Tech Stack

- Playwright
- TypeScript
- Node.js

## Test Coverage

- User login with valid credentials
- Adding product to cart

## How to Run

Install dependencies:

npm install

Run tests:

npx playwright test

Open report:

npx playwright show-report
